🌲 온타리오 캠핑 예약 도우미 가이드

이 가이드는 온타리오 주립공원 예약 시스템에서 원하는 자리를 더 빠르고 정확하게 잡을 수 있도록 도와주는 자동 클릭 스크립트 사용법을 설명합니다.

🤖 이 프로그램은 무엇을 하나요?
  • 시간 동기화: 내 컴퓨터 시간이 아닌, 예약 사이트(Ontario Parks) 서버 시간을 불러와 0.001초 단위로 대조합니다.
  • 카운트다운: 화면 우측 상단에 7시 정각까지 남은 시간을 표시합니다.
  • 자동 클릭: 7시 정각이 되는 순간(오차 약 -0.05초), "Reserve(예약)" 버튼을 자동으로 광속 클릭합니다.

Step 1. 준비하기 (최초 1회)

1. 브라우저(크롬, 엣지 권장)의 북마크바(즐겨찾기 바)가 보이는지 확인하세요.
(안 보인다면: Ctrl + Shift + B 또는 설정에서 '북마크바 표시' 활성화)
2. 아래의 코드를 모두 복사하세요. javascript:(function(){class ResClicker{constructor(target){this.target=target;this.diff=0;this.offset=-50}async sync(){try{const s=Date.now();const r=await fetch('https://reservations.ontarioparks.ca/api/transactionlocation/servertime');const e=Date.now();const lat=(e-s)/2;const st=new Date((await r.text()).replace(/"/g,''));this.diff=(st.getTime()+lat)-e;return true}catch(err){return false}}now(){return Date.now()+this.diff}timer(){const d=document.createElement('div');d.id='rc-t';d.style='position:fixed;top:10px;right:10px;background:black;color:yellow;padding:10px;z-index:9999;font-family:monospace;border:1px solid yellow';document.body.appendChild(d)}click(){const b=document.getElementById('addToStay');if(b)b.click();document.querySelectorAll('.mdc-button__label').forEach(l=>{if(l.textContent.trim()==='Close')l.parentElement.click()})}run(){this.sync().then(ok=>{if(!ok)return;this.timer();const f=()=>{const curr=this.now();const rem=this.target.getTime()-curr;const el=document.getElementById('rc-t');if(el)el.innerText=`REM: ${(rem/1000).toFixed(3)}s`;if(rem<=this.offset){this.click();if(el)el.remove()}else{requestAnimationFrame(f)}};requestAnimationFrame(f)})}}const t=new Date();if(t.getHours()>=7)t.setDate(t.getDate()+1);t.setHours(7,0,0,0);new ResClicker(t).run()})()
3. 북마크바 빈 공간에 마우스 우클릭 -> [페이지 추가]를 누릅니다.
4. 이름을 캠핑 예약 광클(원하는 이름)로 정하고, URL(주소) 칸에 방금 복사한 코드를 붙여넣기 합니다. 저장 버튼을 누르면 준비 완료!

Step 2. 실전 예약하기 (예약 당일)

1. 오전 6시 55분경, 미리 온타리오 공원 사이트에 로그인합니다.
2. 원하는 캠핑장, 날짜, 사이트 번호를 선택하여 "Reserve" 버튼이 보이는 상세 페이지까지 들어갑니다.
(이때 버튼이 비활성화되어 있거나 예약 불가 메시지가 떠 있어도 괜찮습니다.)
3. 오전 6시 59분 00초~30초 사이에, 북마크바에 저장해둔 캠핑 예약 광클 버튼을 한 번 클릭하세요.
4. 화면 우측 상단에 노란색 글씨로 REM: 45.123s와 같은 숫자가 나타나면 정상 작동 중입니다.
5. 손대지 말고 기다리세요. 숫자가 0이 되는 순간 스크립트가 자동으로 "Reserve" 버튼을 누릅니다.
⚠️ 주의사항
  • 로그인 유지: 예약 시도 전 반드시 로그인이 되어 있어야 합니다. 로그인이 풀려 있으면 버튼을 눌러도 로그인 페이지로 이동하게 됩니다.
  • 창 전환 금지: 스크립트 실행 후에는 해당 브라우저 탭을 그대로 띄워두세요. 다른 탭을 보고 있으면 타이머가 느려질 수 있습니다.
  • 새로고침 금지: 스크립트 실행 후 새로고침을 하면 타이머가 사라집니다. 만약 새로고침을 했다면 다시 북마크를 눌러주어야 합니다.

※ 본 프로그램은 예약 성공을 보장하지 않으며, 사용자의 네트워크 환경에 따라 결과가 다를 수 있습니다.
